# hello-world
Hello world project with jode.js using docker.

### Requirements

* Node.js
* Docker
* Docker Compose

### Commmands

* Build and start server: `./build-start-server.sh` 
* View Logs: `./server-logs.sh` 
* Stop server: `./stop-server.sh`
