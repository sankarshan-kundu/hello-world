# hello-world
Hello world project with jode.js using docker.

### Requirements

* Node.js
* Docker
* Docker Compose

## Some useful Docker compose commands

* Build & start stack: `docker-compose up -d --build` 
* View logs: `docker-compose logs -f` 
* Stop stack: `docker-compose down`
